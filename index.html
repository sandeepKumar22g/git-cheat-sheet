Introduction to Git and Github

We are using git CLI => Command Line Interface

Our Git commands

  • git config --global user.name => for checking user name
  • git config --global user.name username => for setting user name
  • git config --global user.email emailId => for setting user email
  • git status => checking the git repo
  • git init => intilaizing the git repo
  • U => means untracked file
  • git add => to add file in our git repo
  • A => means it is added in git repo
  • git add . or git add -A => to add all the untracked files in git repo
  • git commit => then it will open a vi editior after then you have to press i for insert and after you need enter a commit message after that you need to save the message and also need to exit the vi editior => press esc key after => :wq! and hit enter
  • git commit -a -m "messages here" => shoortcut command for the github
  • git remote add origin "link" => add origin
  • git push origin => to upload the changes
  • git push origin master
  • git push --set-upstream origin master => setting the origin git repo
